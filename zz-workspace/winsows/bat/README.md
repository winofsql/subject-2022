- SHIFT_JIS を有効にする
- SET /P A= は環境変数 A に対する入力
```bat

```

## 4) ECHO OFF は、コマンドそのものの表示を止める( バッチファイル側で行う )
```bat
@ECHO OFF 
```
**は、ECHO OFF そのものの表示を止める**
